# 基于 OpenCV 图像相似性判别和处理软件设计


## 综述
图像相似性判别是模式识别、图像分析等领域的重要基础性研究课题，其主要目标是通过一定的相似性判别算法实现两幅或多幅图像之间的同名点检测。

目前，大部分的图像相似性判别算法都是基于图像分辨率相同这一隐含前提，并未考虑分辨率不同或者分辨率未知的情况。为此，本研究希望探究不同分辨率下图像相似性判别算法的准确性和适用性并最终得出不同分辨率下图像相似性的多级判别方法并编程实现相应的处理。

## 预期实现算法

- 直方图判别
- 归一化直方图判别
- 子图像直方图判别
- 哈希算法
- 基于尺度不变性特征(Scale-Invariant Feature Transform, SIFT)

## 实验环境

- Qt 5.15.2 (MSVC 2019, 32 bit)
- OpenCV 4.5.1 x86

## 预期目标

验正所实现算法的正确性或适用性，得出不同分辨率下图像相似性的多级判别方法并编程实现相应的处理。
