## 实验过程

分别测试所选算法在下列情况下的准确性(相似性检验结果大于0.85即85% 则认为相似性较高)：
- 同一幅图像，不同分辨率情况下相似度检验 
- 图像不同但相似的同分辨率和不同分辨率
- 图像不同且不相似的同分辨率和不同分辨率
